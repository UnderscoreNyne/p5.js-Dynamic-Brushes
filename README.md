# p5.js-Dynamic-Brushes
Dynamic Brushes in p5.js

# Mouse and Key Mapping
 
MOUSE
drag              : draw  
 
KEYS  
1-3                 : switch brush  
=                   : increase the radius  
\-                   : decrease the radius  
]                   : increase opacity  
[                   : decrease opacity  
'                   : increase amount of shapes per tick  
;                   : decrease amount of shapes per tick 
f                   : toggle fill  
del, backspace      : erase  
s                   : save png
